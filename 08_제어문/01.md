### 제어문

→ control flow statement

→ 조건문, 반복문

### 블록문

→ block statement / compound statement

→ 0개 이상의 문을 중괄호 묶은 것

→ 코드불록 / 블록

→ 블록문 끝에는 세미콜론을 붙이지 않는다 (언제나 문의 종료를 의미하는 자체 종결성)

### 조건문

→ conditional statement

→ 조건식(boolean으로 평가될 수 있는 표현식)의 평가 결과에 따라 실행을 결정

→ if ... else / switch

### 반복문

→ loop statement

→ 조건식이 거짓일 때까지 반복

→ for / while / do ... while
